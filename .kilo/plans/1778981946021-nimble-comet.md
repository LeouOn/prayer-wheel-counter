# Plan: Juicy Gamification, UI/UX Polish & Encouragement Features

## Guiding Principle

The SPEC says "no XP bars, no achievements with garish badges, no streak-shaming" — the app should feel devotional, not gamified. But there's a rich middle ground: **quiet encouragement, beautiful feedback, and meaningful milestones** that deepen practice rather than cheapen it. Think of it as a sand mandala — intrinsically beautiful, rewarding patience and sustained attention.

---

## Phase 1: Achievement System — "Practice Milestones" (Dignified)

Not XP or badges — but meaningful Buddhist milestones with quiet visual rewards.

### 1.1 Milestone Definitions
Create an `Achievement` system with tiers based on lifetime mantras. Each milestone unlocks something visual/spiritual, not competitive:

| Milestone | Mantras | Reward |
|---|---|---|
| First Rotation | 1 | Welcome animation on first spin |
| Mala Complete | 108 | Wheel skin: Sandalwood (warm brown tones) |
| Thousandfold | 1,000 | Gentle lotus petal particle effect at 1K intervals |
| Ten Thousand | 10,000 | Wheel skin: Lapis Lazuli (deep blue) |
| Hundred Thousand | 100,000 | Wheel skin: Turquoise (Tibetan turquoise) |
| Million | 1,000,000 | Golden aura glow permanently available |
| Ten Million | 10,000,000 | Wheel skin: Ruby (deep red) |
| Hundred Million | 100,000,000 | Stupa silhouette appears on calendar days |
| Billion | 1,000,000,000 | Wheel skin: Amethyst (purple) |
| Trillion | 10^12 | Wheel skin: Rainbow (all colors cycle) |

### 1.2 Implementation
- **New file: `data/model/Achievement.kt`** — Data class + all milestone definitions
- **New file: `data/datastore/AchievementStore.kt`** — Track unlocked achievements in DataStore (Set<String>)
- **New file: `ui/components/MilestoneCelebration.kt`** — Overlay composable shown once when milestone is first reached
  - Subtle, devotional: shows the milestone name in elegant typography, the mantra count, and a brief traditional quote
  - Auto-dismisses after 5 seconds, or tap to dismiss
  - No confetti, no confetti sounds — a single singing bowl tone (or silence)
- **New skins in `WheelSkin.kt`** — Add Sandalwood, Lapis Lazuli, Turquoise, Ruby, Amethyst, Rainbow skins (initially locked)
- **New file: `ui/wheel/AchievementsScreen.kt`** — Grid of all milestones, locked ones shown as silhouettes with the required count, unlocked ones shown in full color with date achieved
- **Update `WheelViewModel.kt`** — Check milestones on `onRotationComplete()` and `loadLifetimeStats()`, fire `triggerMilestoneHaptic()` + show celebration
- **Add Navigation route** — AchievementsScreen accessible from Stats or a new top-bar icon

## Phase 2: "Living" Wheel — More Juicy Visual Feedback

### 2.1 Speed-Dependent Visual Effects
The wheel already has light rays and milestone glow. Enhance:
- **Idle breathing glow** — When stationary, the crystal/finial has a very subtle pulse (0.5s period, 3% alpha oscillation) — the wheel feels "alive" and waiting
- **Speed tiers** — Visual intensity scales with RPM:
  - < 10 RPM: Gentle warmth, faint rays
  - 10-30 RPM: Golden glow intensifies, mantra text becomes more luminous
  - 30-60 RPM: Rainbow rays appear, particles spawn
  - 60+ RPM: Full aurora effect, mandala geometry appears behind wheel (abstract view)
- **Sound of speed** — Optional: RPM-dependent ambient tone (sine wave that rises in pitch, very subtle) — config in audio settings

### 2.2 Enhanced Particle System
Current: Star particles exist but are basic. Enhance:
- **Rotation completion particles** — On each full rotation, emit 2-3 tiny golden motes that drift upward and fade (like incense smoke)
- **Milestone burst** — On power-of-10 milestones, a ring of lotus petals expands outward from the wheel center
- **Session accumulation glow** — The longer you spin in a session, the warmer the background gradient becomes (already partially implemented at `backgroundWarmth` — enhance the range)

### 2.3 Implementation
- **Update `WheelViewModel.kt`** — Add `lastRotationTime` state for particle timing; expose `sessionSpinMinutes` for glow calculations
- **Update all prayer wheel composables** — Add idle breathing glow, speed-tier conditional rendering
- **Update `drawStarParticles`** — Add lotus petal particle type, incense mote type
- Add new `DrawScope.drawLotusPetal()` helper

## Phase 3: Session Enhancement — Encourage Longer, Deeper Practice

### 3.1 Session Intention Cards
Before starting a session, show a rotating "intention card" — a brief Buddhist quote or aspiration:
- 20+ traditional Buddhist aspirations/quotes (stored as string array resource)
- Shown in the IntentionDialog with a "inspire me" button that cycles through them
- The chosen intention is saved and shown on the dedication screen at session end

### 3.2 Session Timer Visualization
When a time goal is set, add a subtle circular progress indicator around the wheel itself:
- A thin ring that fills as you approach the time goal
- When complete, the ring pulses once with the milestone glow
- Non-intrusive — thin enough to not distract from practice

### 3.3 Session Summary Enrichment
The dedication screen (DedicationScreen.kt) currently shows mantras and a text field. Enhance:
- Show session duration, average RPM, peak RPM
- Show how this session compares to your average (e.g., "Your longest session this week")
- Show the gratitude comparison: "In this session you produced what would take X monks Y days to recite"
- Fix the `formatMantras()` in DedicationScreen.kt to use NumberFormatter (currently still has K/M/B-only bug)

### 3.4 Implementation
- **New resource: `res/values/intentions.xml`** — String array of Buddhist quotes/aspirations
- **Update `IntentionDialog.kt`** — Add "inspire me" button with random intention cycling
- **Update `DedicationScreen.kt`** — Accept session stats parameters, show enriched summary, fix formatMantras to use NumberFormatter
- **Update `WheelScreen.kt`** — Pass session stats to DedicationScreen; add circular progress ring around wheel when time goal is active

## Phase 4: Practice Encouragement Features

### 4.1 "Practice Companion" — Gentle Daily Prompts
- A daily notification (opt-in) that shows a Buddhist aspiration + your recent practice stats
- NOT a streak reminder — phrased as inspiration, not obligation
- Examples: "A single rotation carries infinite merit when motivated by compassion. Your lifetime: 2.3B mantras."
- Configurable time in settings (default: off)

### 4.2 Weekly/Monthly Practice Insights
New section on StatsScreen:
- "This week you practiced X% more than last week"
- "Your average session is getting longer" / "You're spinning faster than last month"
- "You've completed more sessions this month than any previous month"
- Quiet comparison to self, never to others

### 4.3 Merit Garden (Long-term feature seed)
A tiny visual garden that grows with practice:
- Each session "plants" a virtual lotus in a small garden view
- Larger sessions = larger flowers
- Garden is personal and contemplative — a visual representation of accumulated practice
- **Defer to Phase 2 planning** — but lay groundwork with data model now

## Phase 5: UI/UX Polish Pass

### 5.1 Fix DedicationScreen Number Formatting
- `DedicationScreen.kt:163-169` still has old K/M/B-only `formatMantras()`. Replace with `NumberFormatter.format()`.

### 5.2 Main Screen Layout Optimization
- The main screen scrolls vertically with many elements stacked. For active spinning, most of the UI should be visible without scrolling:
  - Move CapacitySlider, mantra selector, haptics toggle into a collapsible "Settings" panel at the bottom (swipe up to reveal)
  - Keep the wheel, counter display, and session controls always visible
  - The top bar should auto-hide after 3 seconds of spinning (reappear on tap)

### 5.3 Haptic Patterns
Expand haptic feedback beyond simple vibration:
- **Rotation tick**: Very subtle click (2ms) — feels like a mechanical detent
- **Session start**: Double-tap pattern (10ms + 10ms) — acknowledgment
- **Milestone**: Ascending triple pulse (5ms, 10ms, 20ms with 50ms gaps) — celebratory but dignified
- **Send light**: Gentle wave (already implemented, keep)
- All patterns respect the vibration intensity setting

### 5.4 Wheel Customizer Enhancement
- Show a live preview of the wheel with the selected skin (currently it's just a list)
- Add locked skins for milestone achievements with "Reach X mantras to unlock" label
